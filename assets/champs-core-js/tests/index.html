Remote Select (TomSelect + AJAX + Paginação)

Busca remota com filtros por grupo e paginação por scroll no dropdown. group param default page/per_page

/tests/api/remote_clients.php
Filtros do Grupo clienteFiltro
Se ficar vazio, o request envia unidade_id=1.
Se desmarcar, como existe default=1, o request ainda envia ativo=1. (Remova o default se quiser que desmarcado não envie.)
Vai junto no request: tipo.
Select remoto
Digite 2+ caracteres. Depois, role o dropdown para carregar próximas páginas. O log abaixo deve mostrar page=1, page=2...
(o log espelha as URLs chamadas, ideal para validar defaults e paginação) 


                            

                                Checklist:
                                1) Digitar faz request com q.
                                2) Mudar Unidade/Ativo/Tipo limpa opções e “recomeça”.
                                3) Scroll no dropdown dispara paginação (page sobe).
                                4) Sem TomSelect: aparece console.warn e o select fica nativo.
                            

                        

                    

                


            

        

    


    
    

        

            

                

                    

                        
Validation (CPF / CNPJ / IE)

                        

                            Testes do módulo data-champs-validate com dependências
                            (UF e Documento).
                        

                    

                    champs:validated
                


                


                

                    
                    

                        

                            
Form de Validação


                            

                                
                                

                                    
                                    
                                    

                                        Ao mudar o documento, revalida IE automaticamente.
                                    

                                


                                
                                

                                    
                                    
                                    

                                        Ao mudar UF, revalida IE.
                                    

                                


                                
                                

                                    
                                    
                                    

                                        Regra: IE só valida quando o documento for CNPJ.
                                        Se o doc for CPF, a IE fica neutra (sem erro).
                                    

                                


                                
                                

                                    
                                    
                                    
                                

                            


                            


                            

                                Dica: teste fluxo completo:
                                CPF → IE não deve reclamar;
                                CNPJ + UF → IE deve validar.
